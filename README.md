# 太陽系 · Solar System

一個沉浸式的互動 3D 太陽系網站，用 [Three.js](https://threejs.org/) 打造。整個專案是**單一 `index.html`**——沒有建置流程、沒有外部資源檔，打開就能跑。

## ✨ 特色

- **互動太陽系首頁**：完整的八大行星沿軌道運行，月球繞地球公轉，背景有銀河帶與星空。可自由拖曳旋轉、縮放，閒置時鏡頭自動巡航。
- **點擊進入星球**：點行星聚焦 → 浮出檔案卡 → 進入該天體的完整介紹頁。
- **四個深度內頁**：太陽、地球、月球、火星，各有 7–8 個章節的滾動敘事與歷史故事，配獨立的鏡頭運鏡與配色主題。
- **全程序化 3D 渲染**：每顆星球的表面、大氣、地形起伏都用自訂 GLSL shader 即時生成——沒有用任何貼圖。
  - 太陽：沸騰電漿、太陽黑子、動態日冕
  - 地球：海洋大陸、雲層、極光、衛星群、依緯度的生態帶
  - 火星：沙塵暴、軌道探測器、流星
  - 月球：撞擊坑地形、阿波羅著陸點
  - 木星條紋帶與大紅斑、土星環、天王星傾斜環⋯
- **互動小功能**：
  - 🧮 個人星球檔案換算器（輸入體重/生日，看你在各星球的重量與年齡）
  - 🌍 拖曳直接轉動星球（有慣性）
  - 📏 真實比例模式（一鍵看見行星在宇宙中有多渺小）
  - 📍 3D 地標 pin（hover 看說明、點擊轉到正面）

## 🚀 執行

不需要安裝任何東西，任選一種方式：

```bash
# 方式一：直接用瀏覽器打開
open index.html

# 方式二：起一個本地伺服器
npx serve .
```

> 需要網路連線以載入 Three.js（CDN）與字體。

## 🛠 技術

- Three.js（ES module，CDN 載入）
- 自訂 GLSL vertex / fragment shaders
- 字體：Glow Sans JP、Noto Sans TC、IBM Plex Mono
- 純 vanilla JavaScript，零框架、零建置
